# User CRUD App

A fullstack User Management application built with React (TypeScript) and Node.js (Express + SQLite3).

This project allows you to create, read, update, and delete users with a clean dashboard interface.

---

## Tech Stack

### Frontend

- React
- TypeScript
- Recharts
- Tailwind CSS

### Backend

- Node.js
- Express
- TypeScript
- SQLite3

---

## Setup

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
